# জমা-খরচ ৩৬০ (Income Expense 360)

এটি একটি সম্পূর্ণ ব্যক্তিগত অর্থ ব্যবস্থাপনা মডিউল।

## বৈশিষ্ট্যসমূহ
- ড্যাশবোর্ড: বর্তমান অবস্থা এবং সারসংক্ষেপ।
- লেনদেন: আয় এবং ব্যয়ের হিসাব রাখা।
- রিপোর্ট: চার্ট এবং গ্রাফের মাধ্যমে বিশ্লেষণ।
- সেটিংস: অ্যাপ কনফিগারেশন।

## রক্ষণাবেক্ষণ
- এই ফাইলটি প্রজেক্টের বড় আপডেটের সাথে আপডেট করতে হবে।
